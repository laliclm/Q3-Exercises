HTML

hypertext markup language

- a unique language that is mostly used for coding and developing web pages. it's complicated to use at first but once you're familiar with the tags and attributes you can use it with ease.

- these tags and attributes help on making a variety of interactive elements to be used in websites.

CSS

cascading style sheets

- mainly used for decorating or styling web pages to make it pleasing. it can format html elements to add a unique design or effect for the webpages.

BOOTSTRAP

bootstrap version 4

- an easy-to-use framework to enhance the content of web pages. it also helps on making web pages have a smooth and consistent design on other devices.
